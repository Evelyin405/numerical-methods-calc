# Numerical Methods Calculator

An interactive web-based Numerical Methods Calculator that performs step-by-step computations for:

- Newton–Raphson Method (Root Finding)
- Lagrange Interpolation
- Runge–Kutta 4th Order Method (ODE Solver)

## 🔹 Features
- Step-by-step iteration tables
- Rounding control options
- Clean and user-friendly interface
- Accurate numerical computation

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
